# Lab2 – Интеграционное тестирование

В рамках лабораторной работы 2 в качестве тестируемой системы (SUT) используется
простое веб-приложение на Flask для управления списком задач -> To-do list.

Исходный проект взят с GitHub:
https://github.com/patrickloeber/flask-todo

В каталоге `app/` находится копия приложения, а в каталоге `tests/` интеграционные тесты на Python с использованием `pytest`.
